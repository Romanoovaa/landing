# romanoovaa.ru

Лендинг-сайт психолога Ксении Романовой.

## Структура

- `index.html` — главная страница
- `privacy-policy.html` — политика конфиденциальности
- `cookie-policy.html` — политика cookies
- `terms.html` — условия использования
- `consent.html` — согласие на обработку данных

## Технологии

- HTML
- [Tailwind CSS](https://tailwindcss.com/) (CDN)
- [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) + [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts)

## Деплой

Сайт размещён на GitHub Pages с кастомным доменом `romanoovaa.ru`.
